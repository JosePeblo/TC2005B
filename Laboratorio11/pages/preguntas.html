Describe el archivo package.json

El archivo package.json es un archivo de texto que está escrito en javascript object notation (json) y contiente los nombres de los paquetes que utiliza el proyecto para poder funcionar, scripts para poder correr dependencias como "nodemon" y cualquier otra acción que se quiera automatizar.

Este permite que el código solo se distribuya con los archivos que creamos en el y que al correr npm install se instalen automáticamente todos los paquetes y nuestro código pueda funcionar correctamente

Las dependencias que incluye el archivo vienen con su nombre como llave y la versión de esta como valor, esta versión puede ser estática o puede incuir el caracter de "^" que significa que cuando se vuelva a instalar el proyecto se instalen estos paquetes en su última versión.
